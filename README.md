FakeStoreAPI

Hello and welcome to my final project for React!

This project includes the following:

- Bootstrap Styling
- A homepage with a carousel displaying welcome messages, and introductions to the store and what it sells.
- A product listing page that uses FakestoreAPI to add/delete/edit products
                - Each product shows Image, Title, Price, and a Button to view details
- Product details page that includes detailed information for a single product whenever a product is selected
- A page to add entirely new products of the users chosing
- Error handling
- Stylish interface using Bootstrap
